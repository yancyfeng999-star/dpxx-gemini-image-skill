# Gemini Image User Guide

Use this skill when you want DPXX Gemini image generation through RootFlowAI.

Tell the agent:

- What to draw or edit.
- Model family: `Gemini 3.1 Flash` or `Gemini 3 Pro`.
- Resolution: `1K`, `2K`, or `4K`.
- Ratio: for example `1:1`, `4:5`, `16:9`, or `9:16`.
- Whether reference images are needed.

Default choices:

- Model: `gemini-3.1-flash-image-hd-count`.
- Ratio: choose based on the final use case.

Gemini requests do not include a `quality` parameter.
